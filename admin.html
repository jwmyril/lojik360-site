🛠️ Administration du contenu

Choisissez un tutoriel puis une section : le texte s'affiche comme sur le site et se modifie directement (cliquez dedans, tapez). « Publier » enregistre sur GitHub — le site se met à jour en ~1 minute. Aucune connaissance du code n'est nécessaire.

🔑 Jeton GitHub (nécessaire seulement pour publier)
  1. Ouvrez github.com/settings/personal-access-tokens/new (connecté au compte jwmyril).
  2. Nom : lojik360-admin · Expiration : 1 an · Repository access : « Only select repositories » → lojik360-site.
  3. Permissions → Contents : Read and write. Générez et copiez le jeton.
  4. Collez-le ci-dessous — il reste uniquement dans votre navigateur (localStorage), ne le partagez jamais.
