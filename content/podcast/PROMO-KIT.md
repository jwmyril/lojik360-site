## 1. Mise en place des plateformes (une fois, ~1 h 30)

| Plateforme | Étapes | Astuce |
|---|---|---|
| **Spotify** | creators.spotify.com → nouveau podcast → téléverser MP3 + pochette + description → publier. Récupérer le **flux RSS** (Paramètres → Distribution). | C'est la « source » : tout le reste se branche sur ce RSS. |
| **Apple Podcasts** | podcastsconnect.apple.com → « Ajouter via RSS » → coller le RSS. | Validation en 24–72 h. |
| **YouTube** | Créer la chaîne « Lojik360 » → YouTube Studio → Audio/Podcasts → **« Ajouter un podcast via RSS »** : YouTube crée automatiquement une vidéo (pochette + audio) pour chaque épisode. | Zéro montage vidéo nécessaire pour l'épisode complet. |
| **TikTok** | Compte @lojik360 → on n'y publie PAS l'épisode entier : seulement les **extraits 60–90 s** (voir §3). | 3 extraits par épisode, publiés à J+0, J+2, J+4. |
| **Facebook** | Page « Lojik360 » → post natif à chaque épisode (§4) + partage des extraits vidéo. | Publier la vidéo NATIVEMENT (pas juste un lien) : portée ×3. |
| **WhatsApp** | Créer une **Chaîne WhatsApp** « Lojik360 » → message court + lien à chaque épisode (§5). | Le canal n°1 pour Haïti. |

Quand les liens Spotify/Apple/YouTube existent → **me les envoyer** : je remplace la bannière « Où écouter ? » de podcast.html par les vrais boutons + le lecteur Spotify intégré.
